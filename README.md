
# Indian Strikers Management App

A premier cricket team management application built with React, TypeScript, and Tailwind CSS.

## Features

- **Squad Roster**: Manage player profiles, roles, and availability.
- **Match Scheduling**: Track upcoming fixtures and past results.
- **Match Selection**: Interactive drag-and-drop or click-to-select interface for picking the Playing XI.
- **Fielding Board**: A tactical, interactive cricket field view to plot fielding strategies.
- **Opponent Analysis**: detailed insights into rival teams.
- **Scorecard**: Full match scoring with auto-calculation and summary generation.
- **AI Coach**: Integrated Gemini AI for strategic advice.

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Lucide React (Icons)
- Google Gemini API
- Recharts (Analytics)

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up your `.env` file with `API_KEY` for Gemini integration.
4. Run the development server: `npm run dev`
